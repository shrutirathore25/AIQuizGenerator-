# AI Quiz Generator

Upload a PDF, get an instant multiple-choice quiz — generated by AI, graded like a real exam paper.

Built with **ASP.NET Core (C#)**, styled with a soft "exam paper" aesthetic, and powered by **Groq's LLaMA 3.3 70B** model for question generation.

## Features

- **PDF to Quiz** — upload any PDF and get 5–20 multiple-choice questions generated from its content
- **Adjustable difficulty** — easy, medium, or hard
- **Optional topic focus** — narrow questions to a specific subject within the document
- **Timed quizzes** — live timer while taking the quiz, shown again in the results
- **Flag for review** — mark questions to revisit before submitting
- **Keyboard shortcuts** — press `1`–`4` to answer, `↑`/`↓` to move between questions
- **Smart explanations** — after submitting, each question explains not just the right answer but *why the tempting wrong option is wrong*
- **Downloadable report** — one-click print-to-PDF of your results
- **Quiz history** — recently generated quizzes are remembered locally in the browser

## Tech Stack

- ASP.NET Core MVC (.NET 9+)
- Groq API (`llama-3.3-70b-versatile`) for question generation
- Custom PDF text extraction
- Vanilla JS + CSS (no frontend framework)

## Setup

1. Clone the repo
   ```bash
   git clone https://github.com/shrutirathore25/AIQuizGenerator-.git
   cd AIQuizGenerator-
   ```

2. Get a free Groq API key from [console.groq.com/keys](https://console.groq.com/keys)

3. Copy the example settings file and add your key
   ```bash
   cp appsettings.example.json appsettings.json
   ```
   Then open `appsettings.json` and replace `YOUR_API_KEY_HERE` under `"Groq"` with your real key.

4. Run the project
   ```bash
   dotnet restore
   dotnet run
   ```

5. Open `http://localhost:5200` in your browser

## Project Structure

```
Controllers/       MVC controllers (quiz upload, generation, scoring)
Models/            QuizQuestion and session models
Services/          PDF text extraction, AI question generation, scoring
Views/             Razor views (Upload, Take, Result)
wwwroot/           CSS and JS
```

## Notes

- `appsettings.json` is gitignored — never commit real API keys. Use `appsettings.example.json` as the template.
- Quiz sessions are stored server-side using ASP.NET Core Session, so results persist only for the duration of a browser session.
